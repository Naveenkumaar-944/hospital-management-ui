# рҹҡҖ Getting Started

## Clone Repository

```bash
git clone https://github.com/naveenkumaar-944/hospital-management-ui.git
```

---

## Navigate to Project

```bash
cd hospital-management-ui
```

---

## Install Dependencies

```bash
npm install
```

---

## Start Development Server

```bash
npm run dev
```

---

## Build Project

```bash
npm run build
```

---

## Preview Production Build

```bash

```
